=== Fetch Payout Merkle Tree Path
Fetches payout Merkle tree path by chain ID, asset contract address, Merkle tree root hash and wallet address. the
fetched path can be used to claim payout generated by asset snapshot, see <<Asset Snapshot Request API>>.
